# Fırat Sarıbaş Portfolio

Static GitHub Pages portfolio.

## Deploy

1. Upload these files to the root of the GitHub repository:
   - `index.html`
   - `Firat_Saribas_Portfolio.pdf`
   - `README.md`
2. In GitHub, open **Settings → Pages**.
3. Set source to **Deploy from a branch**.
4. Select branch **main** and folder **/root**.
5. Save and wait for GitHub Pages to publish the site.

The CSS is embedded in `index.html`, so there is no separate stylesheet dependency.
